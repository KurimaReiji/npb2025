# Players

昨シーズン終了後、2024年12月からの支配下選手登録などの情報を「公示」から収集、まとめている。

主な流れは以下の通り。

- 契約保留選手（12月）
- 現役ドラフト（12月）、トレード、FA移籍、ポスティング移籍
- 支配下選手登録（1月末）

シーズン中には、支配下選手登録／抹消、自由契約、トレード、任意引退などがある。

その他、以下のイベントが公示される可能性がある。

- 守備位置変更
- 背番号変更
- 投打変更
- 登録名変更
- 改名
